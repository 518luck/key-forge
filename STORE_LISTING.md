## 详细说明

```
Key Forge 是一款专注隐私的密码生成器。所有密码均在你的浏览器本地生成，绝不联网，不上传任何数据。

【核心功能】

• 密码生成
  - 自定义长度（8–64 位）
  - 自由组合数字、大写字母、小写字母、特殊符号
  - 一键排除易混淆字符（0/O/o/1/l/I 等），复制更省心
  - 自定义排除字符，应对不同网站的符号限制

• 历史记录
  - 本地保存最近生成的密码，随时回看
  - 支持逐条删除与一键清空

• 快捷操作
  - 工具栏点击即用，参数实时调整即时生成
  - 网页任意处右键「生成强密码并复制」，无需打开弹窗
  - 生成完成自动复制到剪贴板

• 主题
  - 浅色 / 深色 / 跟随系统，三态自由切换

• 设置记忆
  - 默认长度、字符集、排除规则、历史条数均可保存

【为什么选择 Key Forge】

密码安全：采用 Web Crypto API（crypto.getRandomValues）配合拒绝采样算法，消除随机数模偏差，生成的每一位都真正不可预测。

零网络：扩展不申请任何网络权限，不连接任何服务器，不存在数据泄露的可能。

全本地：密码、历史记录、设置全部存储在浏览器本地，卸载扩展即彻底清除。

权限最小：仅申请 storage（本地存储）、clipboardWrite（复制）、contextMenus（右键菜单）、notifications（通知）四项必要权限。

【使用场景】

• 注册新账号时生成高强度密码
• 定期更换重要账号密码
• 为不同网站生成符合各自规则的密码
• 临时需要一段随机字符串

【隐私承诺】

Key Forge 不收集、不存储、不传输任何用户数据。没有账号体系，没有云同步，没有统计分析。你生成的密码只属于你。
```

## 权限说明（审核必填）

| 权限 | 用途 |
|------|------|
| storage | 保存用户设置与历史记录到本地 |
| clipboardWrite | 将生成的密码复制到剪贴板 |
| contextMenus | 提供右键「生成密码」快捷入口 |
| notifications | 右键生成后弹出复制成功通知 |
